General Principles:
Systematic Exploration: Search every plausible surface or container exactly once before revisiting; prioritize unopened or unseen locations to cover the whole room methodically.
Immediate Acquisition: As soon as a required object becomes visible and reachable, take it before moving elsewhere to avoid losing track or re-searching.
Destination First Policy: After picking up a goal object, navigate directly to the known target receptacle and place it before resuming further search.
Track Counts & Progress: Maintain an internal counter of how many goal objects are still needed; stop searching or terminate episode only when the counter reaches zero.
Use State-Changing Tools Early: For tasks requiring heated/cooled/clean objects, acquire the object, then immediately use the nearest suitable appliance to change its state before any placement.
Establish Spatial Relations: When a goal specifies a relation (under/inside/on), first locate the reference object, adjust its state if needed (e.g., turn lamp on), then search the specified spatial region for the target object or place it there.

Heat Skills:
Secure Exact Target First: Always identify and pick up the exact object type named in the goal before interacting with the microwave or destination, ignoring look-alikes (e.g., don’t substitute a mug for a cup).
Apply when: After spotting any candidate object or during initial search phase; before opening/using appliances.
Systematic One-Pass Search: Search each plausible surface or closed container once by opening it and inspecting contents; mark searched spots mentally to avoid redundant revisits.
Apply when: While locating the target object and you have not yet found it.
Open Then Heat: Upon reaching the microwave with the target in hand, always open the door, place the object inside, and execute the heat action before leaving.
Apply when: Immediately after navigating to the microwave with the target object held.
No Appliance Before Object: Do not move to or interact with the microwave or final placement location until the target object is picked up, preventing wasted navigation steps.
Apply when: Whenever you are tempted to head to microwave/destination without holding the required object.
Direct Post-Heat Placement: After heating, navigate straight to the specified destination and place the object once, avoiding extra exploration or detours.
Apply when: Right after the heating action completes and the object is in hand.

Mistakes to Avoid:
Don't: Agent repeatedly revisits the same already-searched locations or containers instead of exploring new ones.
Instead: Maintain an exploration map that records which rooms/containers have been fully inspected and prioritize unvisited or unopened candidates until all reasonable options are exhausted.
Don't: Agent decides a closed container is empty without opening it, causing missed discoveries of target objects.
Instead: Always open and inspect every plausible closed container at least once before ruling it out, using a checklist of unopened containers in the current room.
Don't: Agent ignores or walks away from target objects that are already visible and reachable.
Instead: Embed a perception buffer that immediately queues pick-up actions for visible goal objects and interrupts any ongoing exploratory loop.
Don't: Agent keeps searching after acquiring enough items because it loses track of how many targets have been collected.
Instead: Maintain explicit numerical goals and increment/decrement them on every successful pickup or placement, triggering transition to the next sub-goal when counts are satisfied.
Don't: Agent fails to deliver a held target object to the known destination, instead continuing to wander.
Instead: Adopt a hierarchical policy that, upon holding a goal object, immediately switches to a high-priority ‘deliver-to-goal’ subtask before resuming further search.
