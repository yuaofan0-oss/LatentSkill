### Mistakes to Avoid
- **Don't**: Agent repeatedly revisits the same already-searched locations or containers instead of exploring new ones.
  **Instead**: Maintain an exploration map that records which rooms/containers have been fully inspected and prioritize unvisited or unopened candidates until all reasonable options are exhausted. Before submitting a search query, break it down into its key entity and the specific attribute you need.
- **Don't**: Agent decides a closed container is empty without opening it, causing missed discoveries of target objects.
  **Instead**: Always open and inspect every plausible closed container at least once before ruling it out, using a checklist of unopened containers in the current room. If the initial search returns no clear results, rephrase using synonyms or alternate names rather than repeating the same query.
- **Don't**: Agent ignores or walks away from target objects that are already visible and reachable.
  **Instead**: Embed a perception buffer that immediately queues pick-up actions for visible goal objects and interrupts any ongoing exploratory loop. Always cross-check factual claims against at least two independent sources to avoid outdated information.
- **Don't**: Agent keeps searching after acquiring enough items because it loses track of how many targets have been collected.
  **Instead**: Maintain explicit numerical goals and increment/decrement them on every successful pickup or placement, triggering transition to the next sub-goal when counts are satisfied. Before submitting a search query, break it down into its key entity and the specific attribute you need.
- **Don't**: Agent fails to deliver a held target object to the known destination, instead continuing to wander.
  **Instead**: Adopt a hierarchical policy that, upon holding a goal object, immediately switches to a high-priority 'deliver-to-goal' subtask before resuming further search. If the initial search returns no clear results, rephrase using synonyms or alternate names rather than repeating the same query.
